# Content Types & Formats

> Format catalog for the three active platforms. Each format maps to one or more platforms in [[repurpose]]. Select format based on topic (framework vs hot take vs case study) and platform fit.

## Short Take

- **Platforms:** Twitter single, LinkedIn short post
- **When:** Hot takes, observations, single insights, Contrarian Premises
- **Length:** Under 280 chars (Twitter) or 500-800 chars (LinkedIn)
- **Hook type:** Contrarian Premise or Economics Reveal
- **Structure:** Hook + one-sentence proof. No narrative.

## Thread / Multi-Part Post

- **Platforms:** Twitter (3-12 tweets)
- **When:** Step-by-step framework breakdowns, listicles, Economics Reveals with multiple numbers
- **Hook type:** any, but thread-1 must stand alone
- **Structure:** Hook tweet → numbered steps or framework parts → prescription/framework name

## Narrative Post

- **Platforms:** LinkedIn standard or long (1,000-3,000 chars)
- **When:** Case studies, lessons from experience, framework + context
- **Hook type:** Specificity, Credential Anchor, or Framework-First
- **Structure:** Hook → Context → Problem → Reveal → Framework → Application → Question

## Long-Form Article

- **Platforms:** Blog (jcgiusto.com)
- **When:** Full framework pieces, deep Economics Reveals, evergreen references
- **Length:** 1,500-3,000 words
- **Hook type:** Framework-First title or Contrarian Premise title
- **Structure:** see [[blog]] — Title → Opening → Problem → Framework → Implementation → Conclusion

## Case Study (Anonymized)

- **Platforms:** LinkedIn long or Blog
- **When:** Proving frameworks work with real numbers
- **Hook type:** Specificity
- **Structure:** Scenario → Diagnostic → Intervention → Result (numbers) → Framework abstracted
- **Phase 1 constraint:** No Mars Men identifiers. Can reference "a 9-figure subscription brand" but not specific products, team, or proprietary data.

## Deferred Formats (Phase 2+)

- Carousel (LinkedIn documents, Instagram)
- Short-form video (TikTok, Reels, Shorts)
- Long-form video (YouTube)
- Newsletter (1,000-2,000 word weekly email)
